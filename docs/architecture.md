# 系统架构设计

## 整体架构

```
┌─────────────┐  ┌─────────────┐
│  React 前端  │  │  第三方系统  │
│  (Ant Design)│  │  (API对接)  │
└──────┬──────┘  └──────┬──────┘
       │  HTTP/REST     │
       └──────┬─────────┘
              │
┌─────────────▼─────────────────┐
│        API Gateway             │
│    FastAPI + Uvicorn           │
├────────────────────────────────┤
│  ┌──────────┐ ┌──────────────┐ │
│  │ 业务服务  │ │   AI 服务    │ │
│  │  CRUD    │ │  LLM/Embed  │ │
│  └────┬─────┘ └──────┬───────┘ │
└───────┼──────────────┼─────────┘
        │              │
┌───────▼──────────────▼─────────┐
│        PostgreSQL + pgvector    │
│  (结构化数据 + 向量索引)        │
└────────────────────────────────┘
        │
┌───────▼──────────────┐
│       Redis          │
│  (缓存 / 消息队列)    │
└──────────────────────┘
```
