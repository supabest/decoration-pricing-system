# 知识库

供 AI RAG 检索使用的领域知识文档，非结构化知识的集中管理目录。

## 目录说明

| 目录 | 内容 | 用途 |
|------|------|------|
| `benchmark-prices/` | 企业基准价说明、价格编制依据、调价说明文件 | AI 套价时检索基准价规则 |
| `pricing-rules/` | 计价说明、取费规则、套价逻辑说明 | AI 自动计价时参考定价逻辑 |
| `industry-standards/` | 行业规范、国标/省标定额规则、计价规范 | AI 定额推荐的规则依据 |

## 文件格式

- 非结构化知识优先使用 **Markdown** 格式，便于 AI Embedding 分块和检索
- 可包含表格、公式说明、示例
- 文件名按 `领域_主题.md` 命名

## 与 data/ 目录的分工

| 目录 | 存储内容 | 用途 |
|------|---------|------|
| `data/seeds/` | JSON 结构化种子数据 | 数据库初始化导入 |
| `data/price_samples/` | 价格样本数据 | 模型训练或测试 |
| `knowledge/` | 文档型知识 | AI RAG 检索+参考 |
